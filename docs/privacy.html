01

Overview

LinkedIn Post Agent ("the Application") is a personal automation tool developed by an individual developer for the sole purpose of scheduling and publishing content to their own LinkedIn profile using the official LinkedIn API.

This application is not a commercial product, is not available to the public, and does not process the data of any person other than the developer who built and operates it.

02

Data We Collect

This application does not collect personal data from any individual other than the developer's own LinkedIn account credentials obtained through the standard OAuth 2.0 authorization flow.

Specifically, the only data the application accesses is:

OAuth access token
Used to authenticate API requests to LinkedIn on behalf of the account owner.
LinkedIn profile URN
A unique identifier required by the LinkedIn API to specify which account receives the post.
No names, email addresses, profile data, connection lists, messages, or any other LinkedIn data is accessed, read, stored, or transmitted beyond what is strictly required to publish a post.
03

How We Use LinkedIn Data

LinkedIn data is used exclusively for one purpose: publishing scheduled content to the authenticated account owner's own LinkedIn profile via the official LinkedIn UGC Posts API endpoint.

The application does not use LinkedIn data for analytics, advertising, profiling, training machine learning models, or any purpose other than directly posting content as described above.

04

Data Storage & Security

OAuth tokens are stored exclusively as encrypted secrets within GitHub Actions, using GitHub's built-in secret management. They are:

Encryption
Encrypted at rest by GitHub's secret store
Access
Only accessible during workflow execution
Logs
Never printed to logs or console output
Third parties
Never shared with any external service

No external database, analytics service, or third-party storage provider is used by this application.

05

Third-Party Services

This application interacts with the following external services:

LinkedIn API
Used to publish posts. Subject to LinkedIn's Privacy Policy.
GitHub Actions
Used to run the scheduled workflow. Subject to GitHub's Privacy Policy.

No advertising networks, tracking pixels, or analytics services are embedded in this application or this page.

06

AI-Generated Content

Post content is generated using an AI language model based on themes and parameters configured by the developer. All generated content is reviewed and approved by the developer before or as part of the scheduling process. No personal data of any third party is used as input to the content generation process.
